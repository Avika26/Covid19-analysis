# COVID-19 Data Analysis & Visualization

##  Project Overview
End-to-end analysis of global COVID-19 data using Python — covering data cleaning, exploratory data analysis, and interactive visualizations.

##  Project Structure
covid19-analysis/
├── Covid19_EDA.ipynb          # EDA & data cleaning notebook
├── Covid_19_Vis_Dash.ipynb    # Visualization & dashboard notebook
├── owid-covid-data.csv.zip    # Raw dataset (Our World in Data)
├── cleaned_data.csv           # Cleaned output (generated by EDA notebook)
└── README.md

##  What's Inside

### Notebook 1 — EDA & Cleaning
- Missing value handling (fill/ffill strategies)
- Duplicate detection
- Data type fixes
- Logical inconsistency fixes (negative values, deaths > cases)
- Outlier detection using 7-day rolling average
- Export of `cleaned_data.csv`

### Notebook 2 — Visualizations & Dashboard
- Global daily new cases trend
- Top 10 countries by total cases & CFR
- Vaccination rate vs CFR scatter plot
- Correlation heatmap
- Animated choropleth map (Plotly)
- Bar chart race — top 15 countries over time
- Interactive country dashboard (ipywidgets)

##  Dataset
**Source:** [Our World in Data — COVID-19 Dataset](https://www.kaggle.com/datasets/shreyaagupta/covid19-data-by-owid)

##  Libraries Used
`pandas` `numpy` `matplotlib` `seaborn` `plotly` `geopandas` `ipywidgets`

##  How to Run
1. Run `Covid19_EDA.ipynb` first — this generates `cleaned_data.csv`
2. Then run `Covid_19_Vis_Dash.ipynb` for all visualizations
